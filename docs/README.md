# 介绍

一个（仍处于开发初期的）简洁、安全、无后端的静态网站评论系统，基于腾讯云开发。<br>
A simple, safe, serverless comment system based on Tencent CloudBase (tcb).

![Demo](./demo.png)

## 特色

* 评论 | Comment
* 点赞 | Like
* 纯静态 | Static pages
* 可嵌入 | Embedded
* 免费搭建 | Free deploy
* 存储安全 | Security
* 反垃圾评论 | Akismet
* 邮件 / 微信通知 | Email / WeChat notify
